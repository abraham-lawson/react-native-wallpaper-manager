# react-native-wallpaper-manager

react native wallpaper manager

## Installation

```sh
npm install react-native-wallpaper-manager
```

## Usage

```js
import WallpaperManager from "react-native-wallpaper-manager";

// ...

const result = await WallpaperManager.multiply(3, 7);
```
