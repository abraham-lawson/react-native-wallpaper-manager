# react-native-wallpaper-manager

react native wallpaper manager

## Installation

```sh
npm install react-native-wallpaper-manager
```

## Usage

```js
import WallpaperManager, {Type} from "react-native-wallpaper-manager";

// ...

const result = WallpaperManager.setWallpaper(URL, TYPE.FLAG_LOCK)
```
